# 〇×クイズアプリ（Kumite Quiz App）

ブラウザで動作する、CSVベースの〇×クイズ学習アプリです。  
GitHub Pages上で動作し、スマホ・PCのどちらでも快適に利用できます。

---

## 🚀 概要

本アプリは、CSVファイルに登録された問題データをもとに、

- ランダム出題
- 解説表示
- 自動進行
- 要確認チェック

を行う、軽量なクイズ学習ツールです。

特に「短時間で繰り返し学習したい用途」に適しています。

---

## ✨ 主な機能

### ■ 出題設定
- 出題数選択（25 / 50 / 100 / 200問）
- 開始番号選択（例：1, 26, 51...）
- 範囲指定による分割学習が可能

### ■ クイズ機能
- 〇 / × の2択回答
- ランダム出題（重複なし）
- 残り問題数の表示
- 正答数カウント

### ■ 解説表示
- 回答後に即時表示
- 改行を含むCSVにも対応

### ■ 自動進行
- 回答後、一定時間で次の問題へ自動遷移
- 選択可能な時間：
  - 1秒
  - 3秒
  - 5秒

### ■ 要確認チェック
- 「この問題は回答要確認」ボタン
- 後で見直したい問題を記録

### ■ 結果表示
- 正解数 / 総問題数
- 正答率（%）
- 要確認問題一覧表示

## 📄 CSVフォーマット

以下の形式で作成してください。

```csv
number,question,answer,explanation
1,問題文,○,解説文
2,問題文,×,解説文
```

---

## 項目説明

| 項目 | 内容 |
|------|------|
| number | 問題番号（数値） |
| question | 問題文 |
| answer | 正解（○ または ×） |
| explanation | 解説（改行OK） |

---

## 🖥️ 使い方

1. `index.html` をブラウザで開く
2. 出題条件を設定
3. 「開始 / リセット」を押す
4. 〇または×で回答
5. 自動または手動で次の問題へ進む
6. 最後に結果を確認

---

## 🌐 GitHub Pagesでの公開

本アプリは静的サイトのため、そのままGitHub Pagesで公開可能です。

### 手順

1. リポジトリを作成
2. ファイルをアップロード
3. Settings → Pages → Branchを選択
4. URL発行

---

## ⚠️ 制限事項

- データはブラウザ上のみで保持（保存機能なし）
- CSVの書き換えはできない（GitHub Pagesの仕様）
- サーバー機能は使用していない
