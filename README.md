# 〇×クイズアプリ（Kumite Quiz App）

ブラウザで動作する、CSVベースの〇×クイズ学習アプリです。  
GitHub Pages上で動作し、スマホ・PCのどちらでも快適に利用できます。

---

## 🚀 概要

本アプリは、CSVファイルに登録された問題データをもとに、

- ランダム出題
- 解説表示
- 自動進行
- 要確認チェック

を行う、軽量なクイズ学習ツールです。

特に「短時間で繰り返し学習したい用途」に適しています。

---

## ✨ 主な機能

### ■ 出題設定
- 出題数選択（25 / 50 / 100 / 200問）
- 開始番号選択（例：1, 26, 51...）
- 範囲指定による分割学習が可能

### ■ クイズ機能
- 〇 / × の2択回答
- ランダム出題（重複なし）
- 残り問題数の表示
- 正答数カウント

### ■ 解説表示
- 回答後に即時表示
- 改行を含むCSVにも対応

### ■ 自動進行
- 回答後、一定時間で次の問題へ自動遷移
- 選択可能な時間：
  - 1秒
  - 3秒
  - 5秒

### ■ 要確認チェック
- 「この問題は回答要確認」ボタン
- 後で見直したい問題を記録

### ■ 結果表示
- 正解数 / 総問題数
- 正答率（%）
- 要確認問題一覧表示
